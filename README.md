# flower-app

#### 介绍

自研一个花店应用，兼容 app 的微信小程序

#### 软件架构

项目使用 uni-app 框架编译小程序，使用 vue 语法开发，vite 打包 vue 项目

#### 安装教程

1.  安装 node 和 pnpm, 通过 -v 检测是否安装成功
    node -v
    pnpm -v
2.  安装项目的所有依赖
    pnpm i
3.  运行打包
    pnpm dev:app
4.  安装小程序开发工具用来展示打包后的结果

#### 参与贡献

1.  master 主仓库, develop 开发分支
2.  新建 功能分支,
    git branch -b feat/xxx
3.  提交代码
    git push origin feat/xxx
4.  新建 Merge Request
